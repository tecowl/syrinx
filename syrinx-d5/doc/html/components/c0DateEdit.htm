プロパティ

Accessor

published property Accessor: T0DateEditAccessor;
共通したインタフェースで値にアクセスするためのプロパティです。詳しくは、T0DataAccessorクラスのドキュメントを参照してください。  

T0DateEditAccessorクラスには、T0DataAccessorに対して以下のプロパティが追加されています。

published property Year   : Word; //年
published property Month  : Word; //月
published property Day    : Word; //日
published property Hour   : Word; //時
published property Minute : Word; //分
published property Second : Word; //秒
published property MiliSec: Word; //ミリ秒

年/月/日/時/分/秒/ミリ秒のそれぞれの値にアクセスできます。

InputStyle

published property InputStyle: T0CustomDateEditInputStyle;
共通したインタフェースで、文字の入力方法などについて設定するためのプロパティです。詳しくはT0InputStyleクラスのドキュメントを参照してください。

ActivePartKind

K0DateEditItemType = (k0ddtYear, k0ddtMonth, k0ddtDay, k0tdtHour, k0tdtMinute, k0tdtSecond, k0tdtMiliSec);
public property ActivePartKind: K0DateEditItemType;
実行時に年/月/日/時/分/秒/ミリ秒のいづれにフォーカスがあるのかを表します。フォーカスがない場合は、最後の動作を行ったものを表します。

AsDateTime

public property AsDateTime: TDateTime;
実行時に入力されている日付をTDateTime型として表します。

Modified

public property Modified: Boolean;
実行時に値が変更されているかどうかを表します。

AutoAdjustSize

published property AutoAdjustSize: Boolean default True ;
コントロールの大きさを自動的に決めるかどうかを指定できます。デフォルトはTrueで、Trueならば自動的にサイズが決められます。

AutoTab

published property AutoTab: Boolean default True;
入力できる最後の桁を入力し終わったときに、次のコントロールにフォーカスを移動するかどうかを指定できます。デフォルトはTrueです。

EraType

K0EraKind = (k0ekChristian, k0ekJapanese);
published property EraType: K0EraKind default k0ekChristian;
日付の表示を西暦か和暦のどちらで表示するのかを指定します。k0ekChristianが西暦、k0ekJapaneseが和暦です。デフォルトはk0ekChristianです。

JapaneseEra

K0JapaneseEra = (k0jeMeiji, k0jeTaisho, k0jeShowa, k0jeHeisei);
published property JapaneseEra: K0JapaneseEra;
入力されている日付が（西暦ならば和暦に換算して)明治/大正/昭和/平成のどの年号なのかを表します。

k0jeMeiji 明治
k0jeTaisho 大正
k0jeShowa 昭和
k0jeHeisei 平成

メソッド

GetEraChar、GetJEraByChar

K0JapaneseEra = (k0jeMeiji, k0jeTaisho, k0jeShowa, k0jeHeisei);
public class function GetEraChar(JEra: K0JapaneseEra): Char;
public class function GetJEraByChar(AChar: Char): K0JapaneseEra;

GetEraCharメソッドは、K0JapaneseEra型の引数JEraに体操する文字を取得するためのメソッドです。
GetJEraByCharメソッドは、GetEraCharメソッドの逆で文字から該当するK0JapaneseEra型を得るためのメソッドです。
