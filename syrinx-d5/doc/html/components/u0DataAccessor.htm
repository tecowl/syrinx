クラス図

 

T0DataAccessor

コントロールなどの入力されているデータを同じインタフェースで操作するためのクラスです。
使用する側が、コントロールの型をできるだけ気にしないで値の取得/設定することができます。
このクラスを使用している具体的な例として、Tc0Edit、Tc0CheckBoxなどが挙げられます。

主なプロパティ
AsBoolean public property AsBoolean: Boolean; データに論理値でアクセスするためのプロパティです。
AsDateTime public property : TDateTime データにTDateTime型でアクセスするためのプロパティです。
AsFloat public property AsFloat: Double データにDouble型でアクセスするためのプロパティです。
AsInteger public property AsInteger: Integer ; データにInteger型でアクセスするためのプロパティです。
AsString public property AsString: String  データに文字列としてアクセスするためのプロパティです。

具体的にコントロールがどのような値を返すのかは、各コントロールのドキュメントを参照してください。

Header

public property Header: string ;
データを文字列として扱った場合に、HeaderDelimiterプロパティで指定される文字で区切られた先頭の部分にアクセスするプロパティです。

Owner

public property Owner: TComponent;
Accsessorを保持するコンポーネントへの参照です。

HeaderDelimiter

published property HeaderDelimiter: Char default ':';
Headerプロパティによって文字を区切ってアクセスする場合に、どの文字で区切るかを指定するためのプロパティです。

T0StringsAccessor

T0DataAccessorから派生したTStringsをデータとして持ちうるコントロール用のAccessorです。
このクラスを使用している具体的な例として、Tc0ComboBox、Tc0RadioGroup、Tc0Memo、Tc0ListBoxなどが挙げられます。

ItemIndex

public property ItemIndex: Integer;
コントロールがどの項目をさしているのかを示す整数値です。

Source

public property Source: TStrings;
コントロールがプロパティとして保持するTStrings型のインスタンスへの参照です。

LoadEnum

public procedure LoadEnum;

EnumNameプロパティで示すTc0EnumCollectionから項目を読み込みます。
具体的には、EnumNameで示されるTc0EnumCollectionコンポーネントが保持する項目から、Textプロパティ、ItemIDプロパティをすべて読んで、Sourceプロパティに対してAddObjectメソッドを呼び出すことで文字とIDを対応付けます。
また、このメソッドでは自動的にAsIntegerKindプロパティをk0saikFromObjectsに設定します。

AsIntegerKind

K0StringsAsIntegerKind = (k0saikFromItemIndex, k0saikFromString, k0saikFromStringHeader, k0saikFromObjects);
published property AsIntegerKind: K0StringsAsIntegerKind default k0saikFromItemIndex;

AsIntegerプロパティがどのような値を扱うのかを指定するプロパティです。各要素の意味は以下の通りです。

k0saikFromItemIndex ItemIndexをAsIntegerの値として扱います。
k0saikFromString AsStringプロパティを整数値に変換してAsIntegerの値として扱います。
k0saikFromStringHeader Headerプロパティ（ =AｓStringプロパティをHeaderDelimiterプロパティで区切って得られる先頭の部分)を整数値に変換してAsIntegerの値として扱います。
k0saikFromObjects 選択されている文字に関連付けられたObjectsプロパティの参照を、IntegerにキャストしてAsIntegerの値として扱います。

AsStringKind

K0StringsAsStringKind = (k0saskIndexString, k0saskText, k0saskCommaText);
published property AsStringKind: K0StringsAsStringKind default k0saskIndexString;

AsStringプロパティがどのような値を扱うのかを指定するプロパティです。各要素の意味は以下の通りです。

k0saskIndexString Sourceプロパティが保持する文字列のうち、ItemIndexプロパティで指定される文字列をAsStringの値として扱います。
k0saskText SourceプロパティのTextプロパティの値をAsStringの値として扱います。
k0saskCommaText SourceプロパティのCommaTextプロパティの値をAsStringの値として扱います。

EnumName

published property EnumName: string;

LoadEnumメソッドで、文字列とIDのセットを読み込むための、Tc0EnumCollectionコンポーネントを識別する文字列を設定できます。
設定する文字列は、対象となるTc0EnumCollectionコンポーネントのEnumNameプロパティで示される文字列です。
