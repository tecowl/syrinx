共通したインタフェースで、文字の入力方法などについて設定するためのプロパティです。詳しくはT0InputStyleクラスのドキュメントを参照してください。

CurrentExponent

public property CurrentExponent: Integer;

選択されている範囲の最後の位置( = SelStart + SelLength)の位置から、何桁目を指しているのかを表します。

例1）"12345.6789"と入力されている場合

     1      2      3      4      5      .      6       7       8       9      
4 4 3 2 1 0 0 -1 -2 -3 -4

例2) "-1,234,567.890,12"と入力されている場合

      -      1      ,      2      3      4      ,      5      6      7      .      8       9       0       ,       1       2      
6 6 6 5 5 4 3 2 2 1 0 0 -1 -2 -3 -3 -4 -5

TextLength

public property TextLength: Integer;
入力されている文字列の長さを表します。書き込み禁止です。

AsFloat

published property AsFloat: Double;
入力されている数値をDouble型で表します。

AcceptHyphen

published property AcceptHyphen: Boolean default True;
ハイフン"-"の入力を受け付けるかどうかを表します。デフォルトはTrueです。

DecimalLength、IntegerLength

published property DecimalLength: Integer default -1;
published property IntegerLength: Integer default -1;
IntegerLengthプロパティが整数部、DecimalLengthプロパティが小数部に何桁入力できるかを制限します。
それぞれ値が -1 ならば、制限なしと見なします。デフォルトはそれぞれ-1です。

IgnoreZeroOfHead

published property IgnoreZeroOfHead: Boolean default True;
入力されている文字の先頭の0を削除するかどうかを指定します。デフォルトはTrueです。

例)
'0'と入力されていてキャレットが一番最後にあるとき'1'のキーを押下すると、IgnoreZeroOfHead = Trueならば表示は'1'に、
IgnoreZeroOfHead = Falseならば表示は'01'になります。　

ModifyUpDown

published property ModifyUpDown: Boolean default True;
上下キーにて値の変更を行うかどうかを指定できます。Trueならば、文字を１つだけ選択している場合に、上下キーによって値が増減します。デフォルトはTrueです。

ExpoCommaCount、ExpoCommaChar、ExpoCommaWithDecimal、ExpoCommaWithInteger

published property ExpoCommaCount: Integer default 3;
published property ExpoCommaChar: Char default ',';
published property ExpoCommaWithDecimal: Boolean default False;
published property ExpoCommaWithInteger: Boolean default False;

指定した桁数ごとにその区切り文字を入れるかどうか指定するためのプロパティ群です。
ExpoCommaCountプロパティは、何桁ごとに区切るのかを指定します。デフォルトは3です。
ExpoCommaCharプロパティは、桁を区切る際の文字を指定します。デフォルトは","（カンマ）です。
ExpoCommaWithDecimalプロパティは小数部、ExpoCommaWithIntegerプロパティは整数部についてExpoCommaCountプロパティとExpoCommaCharプロパティで指定した区切りを有効にするか否かを指定します。デフォルトはそれぞれFalseです。
