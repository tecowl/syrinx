u0InputStyle

クラス図

目的

文字列入力用コントロールに共通するプロパティ群を同じインタフェースで扱えるようにするためのクラスです。
このクラスのインスタンスは、Tc0EditやTc0ComboBoxなどの文字列を入力できるコントロールのInputStyleプロパティとしてアクセスすることができます。
ReadOnlyやModified、MaxLengthなどVCLに存在するプロパティと、入力文字種の制限、MaxLengthまで入力された場合のフォーカスの移動などを主に扱います。

AcceptCharType、ExclusiveAcceptChar

K1InpCharType = (k1ctAlpha,k1ctDigit, k1ctSign ,   {$IFNDEF NOIMEPROP} k1ctKana ,k1ctKanji {$ENDIF} );
K1InpCharSet = set of K1InpCharType;
published property AcceptCharType: k1InpCharSet
published property ExclusiveAcceptChar: string
AcceptCharTypeプロパティによって、入力できる文字の種類を指定できます。指定できる種類は以下のとおりです。 

k1ctAlpha

アルファベット

k1ctDigit

数字

k1ctSign

記号

k1ctKana

半角カナ

k1ctKanji

全角文字

AcceptCharTypeプロパティのデフォルトは、[k1ctalpha..k1ctkana]です。
ただし、AcceptCharTypeプロパティに指定されない文字種に含まれる文字でも、例外的に入力を許可したい場合、ExclusiveAcceptCharプロパティにその文字を追加することで入力することができるようになります。
ExclusiveAcceptCharプロパティのデフォルトは’’(ヌル文字列)です。

例)
数字と‘-’以外の文字は入力させたくない場合
  XXXX.InputStyle.AcceptCharType := [k1ctDigit];
  XXXX.InputStyle. ExclusiveAcceptChar := ‘-’;

AutoTab

published property AutoTab: Boolean
MaxLengthが0より大きい場合に、入力された文字列の長さがMaxLengthに達したときに、フォーカスを次のコントロールに移動するかどうかを指定するプロパティです。デフォルトはTrueです。

AutoWidth

published property AutoWidth: Boolean
MaxLengthを指定した際に、自動的にコントロールのWidthプロパティを設定するかどうかを指定するプロパティです。デフォルトはTrueです。

Title、TitleDisplay

published property Title: string
published property TitleDisplay: Tc0Label
コントロールがどのようなデータを表す文字列をTitleプロパティは示し、TitleDisplayプロパティはそれをCaptionとして表示するラベルコンポーネントです。

Control

public property Control: TWinControl read GetControl;
T0InputStyleのインスタンスが適用されているコントロールへの参照です。通常、保持しているコントロール自身を参照しています。

CancelAutoTab

public property CancelAutoTab: Boolean;
一時的に、AutoTabプロパティがTrueになっているとしても、それを無効にします。

メソッド

SelectAll、Deselect

public procedure SelectAll;
public procedure Deselect;
入力されている文字列をすべて選択、あるいは選択を解除します。

応用的な使い方

型の分からないコントロールからT0InputStyleを取得する

T0CtrlDistillBroker.GetInputControlメソッドを使用します。

class function GetInputControl(AControl: TObject): T0InputStyle; 

例えば、あるメソッドで引数として渡されるコントロールのReadOnlyを変更したい場合、以下のようなコードで実現できます。 

procedure TForm1.ChangeReadOnly(AControl: TWinControl);
var
  styl: T0InputStyle;
begin
  styl := T0CtrlDistillBroker.GetInputControl(AControl);
  styl.ReadOnly := not styl.ReadOnly;
end;

ただし、AControlに渡されるコントロールがT0InputStyleから派生したクラスのプロパティを持ち、かつそのコントロール用のT0CtrlDistillerの派生クラスが用意されている必要があります。
